NextJS Project

This project was made with NextJS, a React Framework. It's a simple price tracker that uses the NextJS, TailwindCSS, TypeScript, Mongoose (MongoDB), NodeMailer, Axios, Cheerio, React Responsive Carrosel and Google fonts. It's to track the price of a product on a website and send an email when the price drops.
